FUNCTIONS

Functions allow you to group code into reusable blocks, making programs easier to read, write, and maintain. A function can accept inputs, process data, and return outputs. Let's dive into how functions work in Python!

Defining a Function

You define a function in Python using the def keyword, followed by the function name, parentheses (for parameters), and a colon. The code inside the function is indented.

Basic Structure of a Function

def function_name(parameters):
    # code to execute
    return result  # Optional

Example:

def greet(name):
    return "Hello, " + name

print(greet("Alice"))
# Output: Hello, Alice

Function Parameters and Arguments

Functions can accept parameters, allowing them to process different data. Parameters are variables listed in the function definition, while arguments are the values passed to the function when called.

Example with Parameters:

def add(a, b):
    return a + b

print(add(5, 3))
# Output: 8

Default Parameters

Default parameters allow you to specify default values for parameters. If an argument is not passed, the default value is used.

Example:

def greet(name="User"):
    return "Hello, " + name

print(greet())       # Output: Hello, User
print(greet("Bob"))  # Output: Hello, Bob

Return Statement

The return statement ends the function and returns a value to the caller. If there is no return statement, the function returns None by default.

Example:

def square(x):
    return x * x

result = square(4)
print(result)
# Output: 16

Types of Arguments

Python supports various types of arguments in functions, such as positional, keyword, arbitrary, and keyword-only arguments. Here‚Äôs a quick overview:

1. Positional Arguments

Arguments passed by position. The order matters. 

def subtract(a, b):
    return a - b

print(subtract(10, 5))  # Output: 5

2. Keyword Arguments

Arguments can also be passed by explicitly mentioning parameter names. The order doesn‚Äôt matter in this case. 

def multiply(a, b):
    return a * b

print(multiply(b=4, a=3))  # Output: 12

3. Arbitrary Arguments (*args)

Use *args to pass a variable number of positional arguments as a tuple. 

def sum_all(*args):
    total = 0
    for num in args:
        total += num
    return total

print(sum_all(1, 2, 3, 4))  # Output: 10

4. Keyword-Only Arguments (**kwargs)

Use **kwargs to pass a variable number of keyword arguments as a dictionary. 

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(key + ":", value)

print_info(name="Alice", age=25)
# Output:
# name: Alice
# age: 25

Lambda Functions

Lambda functions are small anonymous functions defined with the lambda keyword. They can have any number of parameters, but only one expression.

Example:

double = lambda x: x * 2
print(double(5))
# Output: 10

Summary

  • def - Defines a function.
  • return - Returns a value from the function to the caller.
  • Parameters - Variables listed in the function definition.
  • Arguments - Values passed to the function when called.
  • *args - Accepts variable numbers of positional arguments.
  • **kwargs - Accepts variable numbers of keyword arguments.
  • lambda - Defines an anonymous function.
