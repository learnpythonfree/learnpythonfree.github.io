File Handling in Python

Introduction to File Handling

File handling is an essential part of any programming language. Python provides simple and intuitive ways to work with files. You can read from files, write to files, and manipulate file content easily.

Opening Files

To work with files in Python, you first need to open them. The open() function is used to open files:

# Opening a file for reading (default mode)
file = open('example.txt', 'r')

# Opening a file for writing
file = open('example.txt', 'w')

# Opening a file for appending
file = open('example.txt', 'a')

File Opening Modes

  • 'r' - Read mode (default)
  • 'w' - Write mode (overwrites existing content)
  • 'a' - Append mode (adds to existing content)
  • 'r+' - Read and write mode
  • 'b' - Binary mode (can be combined with other modes)

Reading from Files

Python offers several methods to read file content:

# Reading entire file
file = open('example.txt', 'r')
content = file.read()
file.close()

# Reading line by line
file = open('example.txt', 'r')
for line in file:
    print(line)
file.close()

# Using with statement (recommended)
with open('example.txt', 'r') as file:
    content = file.read()
# File is automatically closed after the block

Writing to Files

You can write content to files using the write() method:

# Writing to a file
with open('output.txt', 'w') as file:
    file.write('Hello, World!')

# Appending to a file
with open('output.txt', 'a') as file:
    file.write('\nNew line of text')

File Methods and Properties

Useful File Methods

  • read() - Read entire file
  • readline() - Read a single line
  • readlines() - Read all lines into a list
  • write() - Write string to file
  • writelines() - Write list of strings to file
  • close() - Close the file

Error Handling

When working with files, it's important to handle potential errors:

try:
    with open('nonexistent.txt', 'r') as file:
        content = file.read()
except FileNotFoundError:
    print('File does not exist!')
except IOError:
    print('An error occurred while reading the file!')
