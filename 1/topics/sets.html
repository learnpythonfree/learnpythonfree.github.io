Python Sets

What are Sets?

Sets are unordered collections of unique elements in Python. They are useful for removing duplicates from sequences and testing membership.

Creating and Using Sets

# Creating a set
fruits = {'apple', 'banana', 'orange'}

# Adding elements
fruits.add('grape')

# Removing duplicates from a list
numbers = [1, 2, 2, 3, 3, 4]
unique_numbers = set(numbers)
print(unique_numbers)  # Output: {1, 2, 3, 4}

Set Operations

Sets support mathematical operations like union, intersection, and difference.

Set Mathematical Operations

# Set operations
set1 = {1, 2, 3, 4}
set2 = {3, 4, 5, 6}

# Union
print(set1 | set2)  # {1, 2, 3, 4, 5, 6}

# Intersection
print(set1 & set2)  # {3, 4}

# Difference
print(set1 - set2)  # {1, 2}
