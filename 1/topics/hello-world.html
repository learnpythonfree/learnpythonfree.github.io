Hello World in Python

Your First Python Program

Welcome to Python programming! The traditional first program in any programming language is the "Hello, World!" program. It's a simple program that displays the text "Hello, World!" on the screen.

Basic Print Statement


print("Hello, World!")

Output: Hello, World!

Understanding the Code

print()

This is a built-in Python function that outputs text to the screen.

"Hello, World!"

This is a string (text) enclosed in quotation marks. Python will display this text exactly as written.

More Print Examples


# Different ways to use print
print("Hello, Python!")              # Prints: Hello, Python!
print('Single quotes work too')      # Prints: Single quotes work too
print("Hello" + " " + "World!")      # Prints: Hello World!
print("Hello", "World!", sep=" ")    # Prints: Hello World!
print("Multiple", "Words", "Here")   # Prints: Multiple Words Here

In Python, you can use either single quotes (') or double quotes (") for strings.

The print() function automatically adds a new line at the end of each print statement.
