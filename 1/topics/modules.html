Python Modules

Importing Modules

Modules are Python files containing functions, classes, and variables that you can use in your programs. Python comes with many built-in modules, and you can also create your own.

Basic Module Usage

# Different ways to import modules
import math
from datetime import datetime
from random import randint, choice
import json as js

# Using imported modules
print(math.pi)  # Using math module
print(datetime.now())  # Using datetime
print(randint(1, 10))  # Using random function
data = js.dumps({"name": "John"})  # Using json with alias

Creating Custom Modules

You can create your own modules by saving Python code in a .py file and importing it into other scripts.

Custom Module Example

# calculator.py
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

def multiply(x, y):
    return x * y

def divide(x, y):
    if y != 0:
        return x / y
    raise ValueError("Cannot divide by zero")

# Using the custom module
from calculator import add, multiply
print(add(5, 3))       # Output: 8
print(multiply(4, 2))  # Output: 8
