Strings in Python

Creating Strings

Strings are sequences of characters in Python. They can be created using single quotes (') or double quotes (").

String Creation


# Different ways to create strings
name = "Alice"
message = 'Hello, World!'
quote = "It's a beautiful day"
multiline = """This is a
multiline string
in Python"""

String Operations

Basic String Operations


# String concatenation
first = "Hello"
last = "World"
greeting = first + " " + last    # "Hello World"

# String repetition
stars = "*" * 3                  # "***"

# String length
text = "Python"
length = len(text)               # 6

# Accessing characters
first_char = text[0]            # "P"
last_char = text[-1]            # "n"

String Methods

Common String Methods


text = "Python Programming"

# Case conversion
print(text.upper())         # "PYTHON PROGRAMMING"
print(text.lower())         # "python programming"
print(text.title())         # "Python Programming"

# Finding substrings
print(text.find("Pro"))     # 7
print("gram" in text)       # True

# Replacing text
new_text = text.replace("Python", "Java")

# Splitting strings
words = text.split()        # ["Python", "Programming"]

String Formatting

Different Ways to Format Strings


name = "Alice"
age = 25

# Using format() method
message1 = "My name is {} and I am {} years old".format(name, age)

# Using f-strings (Python 3.6+)
message2 = f"My name is {name} and I am {age} years old"

# Using % operator (older style)
message3 = "My name is %s and I am %d years old" % (name, age)

# Format specifiers
price = 49.95
formatted_price = f"Price: ${price:.2f}"  # "Price: $49.95"

f-strings (formatted string literals) are the recommended way to format strings in modern Python.

They are more readable and generally faster than other formatting methods.

String Slicing


text = "Python Programming"

# Basic slicing [start:end]
first_word = text[0:6]      # "Python"
second_word = text[7:]      # "Programming"

# Using step [start:end:step]
every_second = text[::2]    # "Pto rgamn"
reverse = text[::-1]        # "gnimmargorP nohtyP"

# Negative indices
last_five = text[-5:]       # "ming"
