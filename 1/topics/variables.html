Variables & Data Types

What are Variables?

Variables are containers for storing data values in Python. Think of them as labeled boxes where you can store different types of information.

Basic Variable Usage


# Creating variables
name = "John"           # String variable
age = 25               # Integer variable
height = 1.75          # Float variable
is_student = True      # Boolean variable

# Printing variables
print(name)            # Output: John
print(age)             # Output: 25
print(height)          # Output: 1.75
print(is_student)      # Output: True

Variable Naming Rules

  • Must start with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Case-sensitive (name and Name are different)
  • Cannot start with a number
  • Cannot use Python keywords (if, for, while, etc.)

Valid vs Invalid Names


# Valid variable names
user_name = "Alice"    # Using underscore
_hidden = "secret"     # Starting with underscore
firstName2 = "John"    # Using numbers (not at start)
CONSTANT = 3.14        # All capitals for constants

# Invalid variable names (will cause errors)
2name = "John"         # Cannot start with number
my-name = "John"       # Cannot use hyphen
class = "Python"       # Cannot use Python keyword

Variable Types

Strings (str)

Text and characters

name = "John"
message = 'Hello'

Integers (int)

Whole numbers

age = 25
count = -10

Floats (float)

Decimal numbers

price = 19.99
pi = 3.14159

Booleans (bool)

True or False values

is_active = True
has_paid = False

Type Conversion

You can convert between different data types using Python's built-in functions:


# String to Integer
age_str = "25"
age_int = int(age_str)    # 25 (integer)

# Integer to String
number = 100
num_str = str(number)     # "100" (string)

# String to Float
price_str = "19.99"
price_float = float(price_str)  # 19.99 (float)

# Integer to Float
num_int = 10
num_float = float(num_int)      # 10.0 (float)
