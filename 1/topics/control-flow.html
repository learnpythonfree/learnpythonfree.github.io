Python Control Flow

If-Else Statements

Control flow statements help you make decisions in your code based on conditions.

Basic If-Else Statement

# Basic if-else statement
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# elif statements
score = 85
if score >= 90:
    grade = 'A'
elif score >= 80:
    grade = 'B'
elif score >= 70:
    grade = 'C'
else:
    grade = 'F'

Loops

Python provides two main types of loops: for loops and while loops.

For Loop

# For loop
for i in range(5):
    print(i)  # Prints 0 to 4

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

# Loop control
for num in range(10):
    if num == 5:
        continue  # Skip 5
    if num == 8:
        break     # Stop at 8
    print(num)
