Dictionaries in Python

Creating Dictionaries

Dictionaries are key-value pair collections in Python. Each key must be unique within the dictionary.

Dictionary Creation


# Different ways to create dictionaries
person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}

# Empty dictionary
empty = {}

# Using dict() constructor
info = dict(name="Alice", age=25)

# Dictionary with mixed value types
mixed = {
    "id": 1,
    "active": True,
    "scores": [85, 90, 95]
}

Accessing and Modifying

Dictionary Operations


person = {"name": "John", "age": 30}

# Accessing values
name = person["name"]      # Using key
age = person.get("age")    # Using get() method

# Modifying values
person["age"] = 31         # Update value
person["city"] = "Boston"  # Add new key-value pair

# Removing items
del person["age"]          # Remove by key
city = person.pop("city")  # Remove and return value
last = person.popitem()    # Remove and return last item

Dictionary Methods

Common Dictionary Methods


person = {"name": "John", "age": 30}

# Getting views
keys = person.keys()           # dict_keys(['name', 'age'])
values = person.values()       # dict_values(['John', 30])
items = person.items()         # dict_items([('name', 'John'), ('age', 30)])

# Updating dictionary
person.update({"email": "john@example.com"})

# Clearing dictionary
person.clear()                 # Remove all items

# Copy dictionary
shallow_copy = person.copy()

Dictionary Comprehension

Dictionary Comprehension Examples


# Create dictionary of squares
squares = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# Filter items
numbers = {'a': 1, 'b': 2, 'c': 3}
evens = {k: v for k, v in numbers.items() if v % 2 == 0}
# {'b': 2}

# Transform values
doubled = {k: v * 2 for k, v in numbers.items()}
# {'a': 2, 'b': 4, 'c': 6}

Dictionary comprehensions provide a concise way to create dictionaries using expressions.

They can include conditions to filter items and transformations for keys and values.

Nested Dictionaries

Working with Nested Dictionaries


# Nested dictionary structure
users = {
    "user1": {
        "name": "John",
        "email": "john@example.com",
        "preferences": {
            "theme": "dark",
            "notifications": True
        }
    },
    "user2": {
        "name": "Alice",
        "email": "alice@example.com",
        "preferences": {
            "theme": "light",
            "notifications": False
        }
    }
}

# Accessing nested values
theme = users["user1"]["preferences"]["theme"]

# Modifying nested values
users["user2"]["preferences"]["notifications"] = True
