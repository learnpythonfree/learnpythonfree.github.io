Operators & Math in Python

Arithmetic Operators

Python provides all the standard mathematical operators for performing calculations.

Basic Math Operations


# Addition
sum = 5 + 3        # 8

# Subtraction
diff = 10 - 4      # 6

# Multiplication
product = 3 * 4    # 12

# Division
quotient = 15 / 3  # 5.0 (always returns float)

# Floor Division
floor_div = 17 // 5  # 3 (rounds down to nearest integer)

# Modulus (remainder)
remainder = 17 % 5   # 2

# Exponentiation
power = 2 ** 3      # 8 (2 raised to power 3)

Assignment Operators


# Simple assignment
x = 10

# Add and assign
x += 5      # Same as: x = x + 5

# Subtract and assign
x -= 3      # Same as: x = x - 3

# Multiply and assign
x *= 2      # Same as: x = x * 2

# Divide and assign
x /= 4      # Same as: x = x / 4

Comparison Operators


# Equal to
5 == 5    # True

# Not equal to
5 != 3    # True

# Greater than
5 > 3     # True

# Less than
3 < 5     # True

# Greater than or equal to
5 >= 5    # True

# Less than or equal to
3 <= 5    # True

Logical Operators


# and operator
True and True     # True
True and False    # False

# or operator
True or False     # True
False or False    # False

# not operator
not True          # False
not False         # True
