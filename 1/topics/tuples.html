Python Tuples

What are Tuples?

Tuples are immutable sequences in Python, similar to lists but cannot be modified after creation. They are often used to group related data together.

Creating and Using Tuples

# Creating tuples
point = (3, 4)
person = ('John', 25, 'New York')

# Accessing elements
print(point[0])      # Output: 3
print(person[1])     # Output: 25

# Tuple unpacking
x, y = point
name, age, city = person

Tuple Operations

While tuples are immutable, they support various operations like concatenation and repetition.

Tuple Operations

# Concatenation
tuple1 = (1, 2, 3)
tuple2 = (4, 5, 6)
combined = tuple1 + tuple2
print(combined)  # Output: (1, 2, 3, 4, 5, 6)

# Repetition
repeated = tuple1 * 2
print(repeated)  # Output: (1, 2, 3, 1, 2, 3)

# Length and counting
print(len(combined))         # Output: 6
print(combined.count(1))     # Output: 1
