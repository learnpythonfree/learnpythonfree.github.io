BASIC OPERATORS

Python provides a variety of operators to perform comparisons, calculations, and logical operations. Understanding these operators is essential for building conditions, performing calculations, and controlling program flow.

Comparison Operators

Comparison operators are used to compare two values. They return True or False based on the condition. These are essential in programming for making decisions and checking conditions.

  • == means Equal
  • != means Not Equal
  • > means Greater than
  • < means Lesser than
  • >= means Greater than or Equal
  • <= means Lesser than or Equal

Example:

print(5 == 8)  # Output -> False
print(5 != 8)  # Output -> True
print(5 > 8)   # Output -> False
print(5 < 8)   # Output -> True
print(5 >= 8)  # Output -> False
print(5 <= 8)  # Output -> True

Arithmetic Operators

Arithmetic operators perform basic mathematical operations. These operators are useful for any kind of calculation, from addition to exponentiation.

  • + means Addition
  • - means Subtraction
  • * means Multiplication
  • / means Division
  • // means Floor Division (division that rounds down to the nearest whole number)
  • ** means Exponentiation (raises one number to the power of another)
  • % means Modulus (returns the remainder after division)

Example:

print(5 + 8)    # Output -> 13
print(5 - 8)    # Output -> -3
print(5 * 8)    # Output -> 40
print(5 / 8)    # Output -> 0.625
print(5 ** 8)   # Output -> 390625
print(5 // 8)   # Output -> 0
print(5 % 8)    # Output -> 5

Logical Operators

Logical operators allow you to combine multiple conditions. Python uses the keywords and, or, and not to perform these operations.

  • and - Returns True if both conditions are True.
  • or - Returns True if at least one condition is True.
  • not - Reverses the Boolean value (if True, it makes it False and vice versa).

Example:

print(10 > 5 and 8 > 3)  # Output -> True (both conditions are true)
print(10 > 5 or 8 < 3)   # Output -> True (one condition is true)
print(not(10 > 5))       # Output -> False (not inverts the true condition)

Summary

  • Comparison Operators compare two values and return True or False.
  • Arithmetic Operators perform mathematical calculations.
  • Logical Operators combine conditions to create complex Boolean expressions.

Example: Putting It All Together

# Combining all operators in one example
a = 10
b = 5
print((a > b) and (a + b > 12))    # Output -> True
print((a > b) or (a + b < 12))     # Output -> True
print(not(a > b))                  # Output -> False
