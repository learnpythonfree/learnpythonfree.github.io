LOOPS

Loops allow you to repeat a block of code multiple times. Python supports two types of loops: for loops and while loops. Let‚Äôs explore each of them!

For Loops

The for loop in Python is used to iterate over a sequence (like a list, tuple, or string) or other iterable objects, executing a code block for each item.

Basic Structure of a for Loop

for item in sequence:
    # code to execute for each item in the sequence

Example:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
# Output:
# apple
# banana
# cherry

Using range() with for Loops

for i in range(5):
    print(i)
# Output:
# 0
# 1
# 2
# 3
# 4

Using range() with Start, Stop, and Step

for i in range(2, 10, 2):
    print(i)
# Output:
# 2
# 4
# 6
# 8

While Loops

A while loop in Python repeats a block of code as long as a specified condition is True. If the condition becomes False, the loop stops.

Basic Structure of a while Loop

while condition:
    # code to execute while condition is True

Example:

counter = 0
while counter < 5:
    print("Counter:", counter)
    counter += 1
# Output:
# Counter: 0
# Counter: 1
# Counter: 2
# Counter: 3
# Counter: 4

Using break and continue in while Loops

Just like in for loops, you can use break and continue statements in while loops.

Using break:

counter = 0
while counter < 5:
    if counter == 3:
        break
    print("Counter:", counter)
    counter += 1
# Output:
# Counter: 0
# Counter: 1
# Counter: 2

Using continue:

counter = 0
while counter < 5:
    counter += 1
    if counter == 3:
        continue
    print("Counter:", counter)
# Output:
# Counter: 1
# Counter: 2
# Counter: 4
# Counter: 5

Using else with while Loops

Similar to for loops, you can use an else clause with a while loop. The else block runs only if the loop exits normally (i.e., without encountering a break statement). 

counter = 0
while counter < 3:
    print("Counter:", counter)
    counter += 1
else:
    print("Loop completed successfully.")
# Output:
# Counter: 0
# Counter: 1
# Counter: 2
# Loop completed successfully.

Summary

  • for - Used for iterating over sequences like lists, tuples, or strings.
  • while - Repeats a block of code while a condition is True.
  • break - Exits the loop entirely.
  • continue - Skips the current iteration and continues with the next.
  • else - Executes if the loop completes normally without a break.
