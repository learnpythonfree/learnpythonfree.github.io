INPUT

The input() function in Python is used to gather user input. It's a simple way to make your programs interactive by asking the user to enter information that your program can use.

Basic Usage of input()

The input() function reads a line of text entered by the user, which it returns as a string. You can optionally provide a prompt to guide the user. 

# Basic input with prompt
name = input("Enter your name: ")
print("Hello, " + name + "!")

Converting Input to Other Data Types

Since input() returns a string, you may need to convert the input to another data type, like an integer or float, using functions like int() or float(). 

# Converting input to an integer
age = int(input("Enter your age: "))
print("You are " + str(age) + " years old.")

Using input() with Calculations

You can use input() to get numbers from users and perform calculations. 

# Getting two numbers and adding them
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))
result = num1 + num2
print("The sum is:", result)

Error Handling with input()

If you expect numeric input, it's a good idea to handle errors in case the user enters non-numeric data. 

# Handling invalid input with try-except
try:
    number = int(input("Enter a whole number: "))
    print("You entered:", number)
except ValueError:
    print("Invalid input! Please enter a number.")

input() in Python 2 vs Python 3

In Python 2, input() automatically tries to evaluate user input, which can lead to security risks. In Python 3, input() is safer, as it treats everything as a string.

Summary of input() Key Points

  • input() is used to take user input as a string in Python 3.
  • Convert the input to other data types (e.g., int, float) as needed.
  • Use try-except blocks to handle errors if expecting specific data types.

Example Summary
