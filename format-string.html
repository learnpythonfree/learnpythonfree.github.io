FORMAT STRINGS

Python provides several ways to format strings, making it easy to include variables or expressions in text output. The most popular and modern method is f-strings, introduced in Python 3.6. We‚Äôll also cover other methods for compatibility.

1. Using F-Strings (Recommended)

F-strings (formatted string literals) are a powerful and easy-to-read way to embed variables or expressions in strings. Just prefix the string with an f and use curly braces { } to insert variables. 

name = "Alice"
age = 25
print(f"Hello, my name is {name} and I am {age} years old.")  
# Output: Hello, my name is Alice and I am 25 years old.

2. Format Method

Before f-strings, the .format() method was commonly used. This method places values into placeholders marked by curly braces { }. 

name = "Alice"
age = 25
print("Hello, my name is {} and I am {} years old.".format(name, age))  
# Output: Hello, my name is Alice and I am 25 years old.

3. Old-Style % Formatting

The % operator was used to format strings in older Python versions. Though less popular today, it‚Äôs still used in some legacy code. 

name = "Alice"
age = 25
print("Hello, my name is %s and I am %d years old." % (name, age))  
# Output: Hello, my name is Alice and I am 25 years old.

Using Expressions in F-Strings

F-strings can handle expressions, making them more versatile. You can even use calculations or method calls within curly braces.

num1 = 10
num2 = 20
print(f"The sum of {num1} and {num2} is {num1 + num2}.")  
# Output: The sum of 10 and 20 is 30.

Formatting Numbers

F-strings allow you to format numbers with options like decimals, alignment, and percentage formatting.

price = 49.99
print(f"Price: ${price:.2f}")  # Display 2 decimal places
# Output: Price: $49.99

score = 0.87
print(f"Score: {score:.1%}")  # Display as percentage
# Output: Score: 87.0%

Multi-Line F-Strings

Use multi-line f-strings to format longer text or strings across multiple lines.

name = "Alice"
age = 25
bio = f"""
Name: {name}
Age: {age}
Welcome to LearnPythonForFree!
"""
print(bio)

'''
Output: Name: Alice
        Age: 25
        Welcome to LearnPythonForFree!
'''

Summary

  • F-strings (e.g., f"Hello, {name}!") are the recommended way to format strings in Python.
  • .format() method is still used in many projects for backward compatibility.
  • Old-style formatting with % is useful in legacy code but generally avoided in new code.

Example: Putting It All Together

name = "Alice"
age = 25
city = "New York"
print(f"{name}, {age}, lives in {city}.")  # Using f-string
print("{}. {} lives in {}.".format(name, age, city))  # Using format method
print("%s, %d, lives in %s." % (name, age, city))  # Using % formatting
