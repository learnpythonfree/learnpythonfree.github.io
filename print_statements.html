PRINT STATEMENT

The print() function is used to display output to the console. It's one of the most commonly used functions, especially for debugging and quickly checking values during code execution. The basic syntax of print() is: 

print(*objects, sep = ' ', end = '\n', file = sys.stdout, flush = False)

  • *objects: These are the values you want to print. They can be strings, numbers, variables, or expressions. Multiple values can be passed separated by commas, and print() will output them in the order they're listed.
  • sep: Specifies the separator between multiple objects. By default, it's a space (' '), but you can set it to any other string.
  • end: Specifies what to print at the end of the output. By default, it‚Äôs a newline character ('\n'), which means print() will move to the next line after executing. You can change it to other characters or an empty string.
  • file: Determines where to send the output. By default, it‚Äôs set to sys.stdout (the console), but you can redirect it to a file.
  • flush: A boolean that, if set to True, forces the output to be written immediately. This is mainly useful in real-time applications where you want to see the output as soon as possible.

Example of Print 

print('Follow me on Instagram and TiKtok')
# Output -> Follow me on Instagram and TiKtok
