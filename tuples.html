TUPLES

A Tuple() is a built-in data structure in Python that is used to store an ordered collection of items. Unlike lists, tuples are immutable, meaning that their elements cannot be modified after creation.

Key Features of Tuples

  • Ordered: Tuples maintain the order of elements.
  • Immutable: Once created, you cannot modify, add, or remove elements.
  • Indexed: Each element in a tuple has an index for access.
  • Allow Duplicates: Tuples can contain duplicate elements.

Creating a Tuple

You can create a tuple by enclosing elements in parentheses (), separated by commas.

# An empty tuple
my_tuple = ()

# A tuple with elements
my_tuple = (1, 2, "apple", 3.5)

Accessing Elements in a Tuple

Tuples use zero-based indexing, meaning the first item has an index of 0.

# Accessing the first element
print(my_tuple[0])  # Output: 1

# Accessing the last element
print(my_tuple[-1])  # Output: 3.5

Tuple Packing and Unpacking

Tuples support packing and unpacking, allowing you to assign multiple values in a single line. 

# Packing values into a tuple
packed_tuple = 1, "apple", 3.5

# Unpacking tuple values
a, b, c = packed_tuple
print(a)  # Output: 1
print(b)  # Output: "apple"
print(c)  # Output: 3.5

Common Tuple Operations

Length of a Tuple

Use len() to get the number of elements in a tuple.

my_tuple = (1, "apple", 3.5)
print(len(my_tuple))  # Output: 3

Checking if an Element is in the Tuple

Use the in keyword.

if "apple" in my_tuple:
    print("Apple is in the tuple!")

Slicing Tuples

Slicing allows you to get a subset of the tuple.

numbers = (1, 2, 3, 4, 5)
print(numbers[1:4])  # Output: (2, 3, 4)
print(numbers[:3])   # Output: (1, 2, 3)
print(numbers[2:])   # Output: (3, 4, 5)

NOTE! IN numbers[1:4] 1 IS INCLUDED WHILE THE INDEX 4 IS NOT INCLUDED!

Tuple Methods Summary

count(): Counts the occurrences of a specific element
index(): Returns the index of the first occurrence of an element

Example: Summary

# Creating and accessing elements in a tuple
my_tuple = (1, "banana", 3.5)
print(my_tuple[0])  # Output: 1

# Packing and unpacking
person = ("Alice", 25, "Engineer")
name, age, profession = person
print(name)  # Output: Alice
print(age)   # Output: 25

SUMMARY

  • Tuples are ordered and immutable collections.
  • They are useful for fixed data that shouldn't change.
  • They support basic operations like slicing, packing/unpacking, and indexing.
