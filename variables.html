VARIABLES

A variable is a symbolic name that is a reference or pointer to an object. Once a variable is assigned a value, it points to the object holding that value. Variables in Python do not need an explicit declaration to reserve memory space, as the assignment operation automatically binds the variable to the data. IN SIMPLE WORDS variables are like container which you can store strings(alphabets), integer(numbers), boolean(True or False), etc... 

a = 82 
print(a)
# Output 82

Why Use Variables?

Variables are important because they allow you to: Store data that can change while your program is running (like a score in a game). Make your code easier to read and manage. Avoid repeating the same value multiple times in your code. Naming Variables: You can name a variable almost anything you want, but there are some rules: It must start with a letter (a-z, A-Z) or an underscore (_). It can contain letters, numbers (0-9), and underscores, but no spaces. It cannot start with a number. Variable names are case-sensitive (e.g., myVariable and myvariable are different). You can assign a value to a variable using the = sign. Like the example You saw above. Make sure to learn about Comments
