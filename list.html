LISTS

A List() is a built-in data structure that is used to store an ordered collection of items. Lists are one of the most versatile and commonly used data types in Python because they allow you to work with a collection of values in a single variable.

Key Features of Lists

  • Ordered: Lists maintain the order of elements. If you add items in a certain order, they will stay in that order.
  • Mutable: Lists can be changed after they're created. You can add, remove, or modify elements in a list.
  • Indexed: Each element in a list has an index, allowing you to access individual items.
  • Allow Duplicates: Lists can contain duplicate elements.
  • Dynamic Size: Lists can grow and shrink as you add or remove elements.

Creating a List

You can create a list by enclosing elements in square brackets [ ], separated by commas.

# An empty list
my_list = []

# A list with elements
fruits = ["apple", "banana", "cherry"]

Accessing Elements in a List

Lists use zero-based indexing, meaning the first item has an index of 0.

# Accessing the first element
print(fruits[0])  # Output: apple

# Accessing the last element
print(fruits[-1])  # Output: cherry

Modifying Lists

Since lists are mutable, you can change the elements within them.

Changing Elements

You can modify an element by referring to its index.

fruits[1] = "blueberry"
print(fruits)  # Output: ["apple", "blueberry", "cherry"]

Adding Elements

  1. append() - Adds an element to the end of the list.
  2. insert() - Adds an element at a specified position.
# Adding an item to the end
fruits.append("orange")
print(fruits)  # Output: ["apple", "blueberry", "cherry", "orange"]

# Inserting an item at index 1
fruits.insert(1, "banana")
print(fruits)  # Output: ["apple", "banana", "blueberry", "cherry", "orange"]

Removing Elements

  1. remove() - Removes the first occurrence of a specific element.
  2. pop() - Removes an element by its index and returns it. If no index is provided, it removes the last item.
  3. clear() - Removes all elements, making the list empty..
# Removing a specific element
fruits.remove("banana")
print(fruits)  # Output: ["apple", "blueberry", "cherry", "orange"]

# Removing the last item
last_item = fruits.pop()
print(last_item)  # Output: orange
print(fruits)     # Output: ["apple", "blueberry", "cherry"]

# Removing an item by index
fruits.pop(1)
print(fruits)  # Output: ["apple", "cherry"]

# Clearing all items
fruits.clear()
print(fruits)  # Output: []

Common List Operations

Length of a list

Use len() to get the number of elements in a list.

fruits = ["apple", "banana", "cherry"]
print(len(fruits))  # Output: 3

Checking if an Element is in the List

Use the in keyword. 

if "apple" in fruits:
print("Apple is in the list!")

Slicing Lists

Slicing allows you to get a subset of the list.

numbers = [1, 2, 3, 4, 5]
print(numbers[1:4])  # Output: [2, 3, 4]
print(numbers[:3])   # Output: [1, 2, 3]
print(numbers[2:])   # Output: [3, 4, 5]

Sorting a List

You can sort a list in ascending or descending order.

numbers = [5, 2, 9, 1, 5, 6]
numbers.sort()
print(numbers)  # Output: [1, 2, 5, 5, 6, 9]

numbers.sort(reverse=True)
print(numbers)  # Output: [9, 6, 5, 5, 2, 1]

List Comprehension

List comprehension is a concise way to create lists.

# Create a list of squares
squares = [x**2 for x in range(1, 6)]
print(squares)  # Output: [1, 4, 9, 16, 25]

List Methods Summary

append(): Adds an element to the end of the list
insert(): Adds an element at a specific index
remove(): Removes the first occurrence of a specific element
pop(): Removes an element by index and returns it
clear(): Removes all elements from the list
index(): Returns the index of the first occurrence of an element
count(): Counts occurrences of a specific element
sort(): Sorts the list in ascending or descending order
reverse(): Reverses the order of the list
copy(): Returns a shallow copy of the list

  • Lists are ordered, mutable, and indexed collections.
