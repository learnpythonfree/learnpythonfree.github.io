HELLO WORLD

In Python, the "Hello, World!" program is as simple as it gets. You can print "Hello, World!" to the console with just one line: 

print("Hello, World!")
# Output Hello, World!

The print() function is a built-in function in Python that outputs data to the console or standard output device. It can take various arguments and has several parameters that allow you to control its behavior. The print function is not a reserved keyword; you can technically create a variable named print (though it is not recommended because it would overshadow the built-in function). This distinction between functions and keywords is crucial in understanding how Python works and how to write effective code. Don't Worry you will know about this as you learn python.
