IF-ELSE STATEMENTS

Conditional statements like if, else, and elif allow you to control the flow of your code based on certain conditions. They enable you to create decision-making structures in your programs.

Basic Structure of an if Statement

The if statement checks a condition and executes a block of code only if the condition is True. The general syntax for an if statement is: 

if condition:
    # code to execute if condition is True

Example:

age = 18
if age >= 18:
    print("You are eligible to vote!")  # Output -> You are eligible to vote!

Adding an else Statement

The else statement provides an alternative code block that runs when the if condition is False. The syntax is: 

if condition:
    # code to execute if condition is True
else:
    # code to execute if condition is False

Example:

age = 16
if age >= 18:
    print("You are eligible to vote!")
else:
    print("You are not eligible to vote.")  # Output -> You are not eligible to vote.

Using elif (else-if)

elif allows you to check multiple conditions. If the first if condition is False, elif checks another condition, and so on. You can have as many elif statements as needed. 

if condition1:
    # code to execute if condition1 is True
elif condition2:
    # code to execute if condition2 is True
else:
    # code to execute if all above conditions are False

Example:

score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")  # Output -> Grade: B
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

Nested if Statements

You can place an if statement inside another if statement. This is known as "nesting" and is useful when you need to check additional conditions within a condition. 

if condition1:
    if condition2:
        # code to execute if both conditions are True
    else:
        # code to execute if condition1 is True and condition2 is False
else:
    # code to execute if condition1 is False

Example:

age = 20
citizenship = "yes"
if age >= 18:
    if citizenship == "yes":
        print("Eligible to vote")  # Output -> Eligible to vote
    else:
        print("Not eligible to vote (citizenship required)")
else:
    print("Not eligible to vote (age requirement not met)")

Logical Operators with if Statements

You can combine multiple conditions in an if statement using logical operators like and, or, and not. This allows for more complex decision-making. 

if condition1 and condition2:
    # code to execute if both conditions are True

if condition1 or condition2:
    # code to execute if at least one condition is True

if not condition:
    # code to execute if condition is False

Example:

temperature = 25
humidity = 80
if temperature > 20 and humidity > 70:
    print("It's a warm and humid day")  # Output -> It's a warm and humid day

if temperature > 30 or humidity > 70:
    print("Stay hydrated!")  # Output -> Stay hydrated!

Summary

  • if - Checks a condition and executes code if the condition is True.
  • else - Provides an alternative code block if the if condition is False.
  • elif - Allows for additional conditions if the if condition is False.
  • Nested if - if statements within other if statements for more complex checks.
  • Logical Operators - Used to combine multiple conditions with and, or, and not.

Example: Combining It All Together

# Checking grade and other conditions
grade = 88
passed = True

if grade >= 90:
    print("Excellent performance!")
elif grade >= 80 and passed:
    print("Good job, you passed!")  # Output -> Good job, you passed!
else:
    print("Try harder next time!")
